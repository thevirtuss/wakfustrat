{% trans 'À propos' %}

{% trans 'WakfuStrat, kesako ?' %}

{% blocktrans %}WafkuStart est un site regroupant des stratégies pour le jeu WAKFU, un MMORPG édité par Ankama. Il s'agit d'un site non-officiel sans aucun lien avec Ankama.{% endblocktrans %}

{% blocktrans %}Nous sommes partis d'un constat simple : la majorité des précédents fan-sites sur Wakfu sont laissés à l'abandon par leurs créateurs. De plus, il était vraiment difficile pour certains d'apporter des modifications aux guides déjà éxistant pour signaler une erreur. De ce constat est né WakfuStrat, projet dans les cartons depuis plusieurs années.{% endblocktrans %}

{% trans 'Comment est financé le site ?' %}

{% blocktrans %}Tout d'abord il est important de préciser que le site est entièrement gratuit, sans publicité ET LE RESTERA, nous nous y engageons. Nous ne revendons aucune donnée personelle, nous n'utilisons pas de service tier pour vous pister.{% endblocktrans %}

{% blocktrans %}Actuellement le site est financé entièrement par l'équipe qui le développe. Cela représente environ 50€ par an, comprenant le nom de domaine et la location du serveur. Dans un futur proche nous mettrons en place plusieurs solutions pour que vous puissiez nous aider à financer le site.{% endblocktrans %}
